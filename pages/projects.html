Student Enrollment Application

Description: Through a drag and drop feature on a web-based GUI, the application processes 24 Excel files containing student information from the 24 districts Ulster BOCES works with, stores the information from the Excel files to a Microsoft SQL server, and generates an Excel file containing enrolled students for each course per district, eliminating the manual enrollment process.

Technologies and Implementation:

  • Integrated MS SQL Server database using Java JDBC for reliable data retrieval.
  • Constructed SQL queries in Java and executed them using java.sql classes to perform CRUD operations in a MS SQL Server database.
  • Utilized JavaFX classes such as WebView and WebEngine to render the web-based GUI, and JSObject to call Java functions from JavaScript and vice versa to dynamically update the UI based on user actions.
  • Utilized Apache POI libraries such as Workbook and XSSFWorkbook to manipulate Excel files programmatically.

IN PROGRESS: Roommate Matching App

Description: I'm making an app built in Flutter that helps users find roommates either in their preferred state to move to, or nearby where they reside. Users will match with people that have similar interests/preferences. Preferences are created on a user's profile. The more complete a user's profile is, the more accurate their matches will be.

Technologies and Implementation:

  • Implementing Supabase edge functions to moderate user profile images using Google's Cloud Vision API and upload images to Supabase storage buckets.
  • Using Firebase for user account creation and email verification, and Cloud Firestore to store user information.
